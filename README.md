## 1. 项目简介

- #### 1.1 技术选型
      微服务组件：Spring Cloud Alibaba(注册中心/配置中心：Nacos、限流/降级/熔断：Sentinel)、
                 Spring Cloud(服务调用：OpenFeign、客户端负载：内嵌的Ribbon)
                 网关：Nginx + Spring Cloud Gateway
      
      各服务框架：SpringBoot
 
      持久层框架：MyBatis Plus
      
      安全框架：Spring Security
 
      数据库：MySQL
 
      缓存：Redis
